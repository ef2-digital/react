## Add Icons

To add more icons, use the theme provider.

### Next.js

```tsx
// theme.ts
import { PartialTheme } from '@ef2-digital/react';
import HomePin from '!@svgr/webpack!@material-symbols/svg-600/rounded/home_pin.svg';

const theme: PartialTheme = {
    homePin: HomePin
};

export default theme;
```

```tsx
// Inside a component.

<Icon name="homePin" />
```

### Color
```tsx
<Icon name="homePin" className="fill-green-600" />
```
